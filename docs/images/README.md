# Imagenes Para El README

Pon aqui las capturas que quieres mostrar en el `README.md`.

Nombres actualmente usados en el README:

- `cover.png`
- `home.png`
- `chat.png`
- `avatar-chat.png`
- `avatar-cha overclok.png`
- `explore.png`
- `game.png`
- `game biblia fragmentada.png`
- `profile.png`

Si quieres dejarlo mas limpio despues, te conviene renombrar:

- `avatar-cha overclok.png` -> `avatar-overclock.png`
- `game biblia fragmentada.png` -> `game-biblia-fragmentada.png`

Formato recomendado:

- `png`
- ancho entre `1080` y `1600` px
- misma proporcion visual entre capturas para que la tabla del README se vea limpia

Si quieres una portada horizontal, usa:

- `cover.png`

Si no existe una imagen, GitHub mostrara el nombre roto en el README. Por eso conviene agregar al menos las capturas principales antes de publicar el repositorio.
